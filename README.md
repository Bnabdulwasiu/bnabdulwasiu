## 📈 What I've Learned the Hard Way

- **Database durability** isn't magic — it's append-only logs preventing byte-level corruption
- **Connection pooling + indexing** dropped query times from 320ms → <5ms without extra infra
- **Streaming CSV parsing** (1,000-record chunks) prevents OOM crashes on million-row datasets
- **UTF-8 byte lengths ≠ character lengths** — a lesson learned from a production offset bug
